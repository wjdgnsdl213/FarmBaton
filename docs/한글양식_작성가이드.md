## 0. 준비물
- 공식 양식 파일: `02_경진대회_기획서_팜바톤.hwp`

- 첨부물: `docs/팜바톤_포트폴리오.pdf`, 시연 영상(링크 또는 파일)

## 1. 섹션 매핑 (초안 → 양식 칸)
양식의 항목 제목은 그대로 두고, 아래 초안 섹션 내용을 해당 칸 본문에 붙여 넣습니다.

| 양식 칸 | 초안 섹션 |
|---------|-----------|
| 제품 및 서비스명 | 1 |
| 공모 분야 | 2 (☑ 웹) |
| 등록 정보 | 3 (웹 URL + GitHub) |
| 서비스 개요 (300자) | 4 |
| 개발 서비스의 기능 및 특징 | 5 |
| AI 기술 활용 여부 | 6 |
| 기존 서비스와의 차별성 및 독창성 | 7 |
| 제품 및 서비스 사업성 | 8 (팀·수익모델·시연 시나리오) |
| 농림축산식품 공공데이터 활용 적정성 | 9 (데이터 표) |
| 기타 | 10 |
| 기대효과 | 11 |
| 제품 및 서비스 개발 실적 | 12 |

## 2. 작성 순서 팁 (배점 높은 순서로 공들이기)
1. **9 공공데이터 적정성 + 6 AI 활용 (합 30점)** — 가장 먼저, 가장 꼼꼼히. 시스템명·URL·데이터명 표를 정확히.
2. **5 기능·특징 (20)** — 화면 캡처 3장(input/report/match) 삽입.
3. **7 차별성 (20)** — 비교 표 그대로.
4. **8 사업성 (20)** — 팀 표 + 수익모델 + 시연 시나리오.
5. **4 개요·11 기대효과 (10)** — 통계 포함.

## 3. 표 옮기기
- 마크다운 표(`| ... |`)는 한글에서 **표 그리기**(Ctrl+N,T)로 새로 만들어 셀에 값만 옮기는 게 가장 깔끔합니다.
- 데이터 출처 표(9번)·팀 표(8번)·비교 표(7번)가 대상.

## 4. 이미지 삽입
- 한글: **입력 → 그림**(Ctrl+N,I)로 `docs/portfolio/img/`의 PNG 삽입.
- 5번(기능)에 input→report→match 3장을 흐름으로, 8번(시연 시나리오)에 1~2장.
- 캡처가 더 필요하면 production에서 추가 촬영.

## 5. 공공데이터 출처 캡처 첨부 (가점)
아래 7건을 캡처해 양식 9번 근처 또는 별도 첨부:
1. V-World 인증키 발급 화면 (vworld.kr 마이페이지)
2. 팜맵 (agis.epis.or.kr)
3. 농산물 소득조사 (nongsaro.go.kr / 공공데이터포털)
4. KAMIS 가격·평년가 (kamis.or.kr)
5. 개별공시지가 (공공데이터포털 상세페이지 — URL·데이터ID 보이게)
6. 토지 실거래가 (rt.molit.go.kr)
7. 법정동코드 (공공데이터포털 상세페이지)
