## [복붙용 본문]

팜바톤(FarmBaton)은 고령 농가가 보유한 농장 자산과 영농 기반을 청년농에게 연결하는 농장 승계 진단·매칭 서비스입니다. 서비스의 핵심은 농장주에게는 “내 농장을 어느 정도 조건으로 넘길 수 있는지”를 공공데이터 기반으로 빠르게 보여주고, 청년농에게는 “내 자본·작목·지역 조건에 맞는 승계 가능 농장이 어디인지”를 점수순으로 추천하는 것입니다. MVP는 충북·경북·충남 3개 도와 사과·복숭아·포도 3개 과수 작목에 집중해 실제 데모 가능한 범위로 구현했습니다.

서비스는 두 가지 주요 흐름으로 구성됩니다.

첫째, 농장주용 “농가 등록 → 인수 검토 리포트” 흐름입니다. 농장주는 회원가입 후 주소를 입력하고 작목, 과수 수령, 희망 승계 방식, 시설·판로·매출 정보를 단계적으로 입력합니다. 주소는 V-World 지오코딩을 통해 좌표로 변환되고, 팜맵 기반 공간연산으로 필지 위치와 면적을 자동 취득합니다. 이후 개별공시지가, 토지 실거래가 적재본, 농산물 소득조사, KAMIS 시세, 시설 표준단가, 수령계수 등을 결합해 예상 연소득과 “인수 검토가 범위(참고용 추정)”를 산출합니다. 결과 화면은 토지 기준가, 시설 잔존가, 영업권, 예상 연소득, 신뢰등급(A~D), 산출 근거를 함께 보여주며, 단일 가격이 아니라 min~max 범위로 제시합니다. 시설 자료, 매출 자료, 판로 정보가 많을수록 신뢰등급이 올라가도록 설계해 농장주가 추가 자료를 제출할 유인을 제공합니다.

둘째, 청년농용 “프로필 등록 → 매칭 리스트” 흐름입니다. 청년농은 희망 지역, 희망 작목, 보유 자본, 영농 경력, 정책자금 활용 여부, 선호 승계 방식을 입력합니다. 팜바톤은 지역 적합도, 작목 일치도, 자본 충족도, 경험 수준, 승계 방식 적합도, 정책자금 조건 등 6개 요인을 100점 만점으로 계산해 승계 가능 농장을 점수순으로 추천합니다. 청년농은 농장별 예상 연소득, 인수 검토가 범위, 시설 현황, 맞춤 지원사업, 관점별 리포트를 확인한 뒤 상담을 신청할 수 있습니다. 농장주가 상담 요청을 수락하면 인앱 채팅으로 이어져 실제 승계 논의를 시작할 수 있습니다.

서비스의 기술 구조는 신뢰성과 데모 안정성을 우선해 설계했습니다. 백엔드는 FastAPI와 PostgreSQL/PostGIS 기반이며, 팜맵 SHP 등 공간데이터는 EPSG:4326으로 통일해 저장합니다. 프런트엔드는 React(Vite)와 Leaflet 지도를 사용한 모바일 퍼스트 반응형 웹입니다. 외부 API 장애가 발생해도 서비스가 멈추지 않도록 모든 외부 호출에는 정적 폴백을 두었고, 실거래가는 API 실시간 호출 없이 DB에 적재한 정적 데이터만 사용합니다. 단가·계수·기준값은 코드에 하드코딩하지 않고 DB 기준 테이블을 참조합니다.

AI는 설명 생성에만 사용합니다. 금전 추정치와 매칭 점수는 모두 결정론적 Python 함수로 계산해 동일 입력에 동일 결과가 나오도록 했습니다. 생성형 AI(Claude)는 계산된 수치를 바꾸지 않고, 농장주에게는 승계 준비와 농장 가치 설명 관점으로, 청년농에게는 자본 회수 가능성·주의사항·정책자금 활용 관점으로 리포트를 풀어 쓰는 역할만 담당합니다. 즉, “숫자는 알고리즘, 설명은 AI”로 분리해 환각 위험을 줄이고 설명의 이해도를 높였습니다.

팜바톤은 단순 농지 매물 서비스가 아니라 농지·과수·시설·판로를 묶은 영농 기반 전체의 승계를 다룹니다. 고령 농가는 복잡한 데이터와 평가 과정을 거치지 않고도 무료로 사전 검토 리포트를 받을 수 있고, 청년농은 막연한 탐색 대신 자신의 조건에 맞는 농장을 우선순위로 확인할 수 있습니다. 현재 production 환경(farmbaton.vercel.app)에서 농가 등록, 리포트 생성, 청년농 매칭, 상담 신청, 채팅까지 핵심 흐름이 실제 동작하도록 구현되어 있습니다.
