# 팜바톤 시연 영상 시나리오 (약 2분 30초)

> 목적: 제출용 데모 영상. 평가 최고 배점인 **공공데이터·AI 활용(30)** 이 화면에서 분명히 드러나게 구성.
> 데모 계정: 농장주 `demo.farmer@farmbaton.kr` / 청년농 `demo.young1@farmbaton.kr` (비번 `farmbaton!2026`).
> production: https://farmbaton.vercel.app

## 사전 준비
- 화면녹화: OBS Studio 또는 Windows `Win+G`(게임바). 해상도 1920×1080, 마우스 클릭 강조 ON.
- 브라우저 전체화면(F11), 북마크바 숨김. 인트로 애니메이션을 보여주려면 시작 전 새 시크릿창(인트로가 세션당 1회라).
- **파이(V-World 프록시) 켜두기** — 임의 주소 실시간 지오코딩 장면을 위해 필수.
- 멘트는 직접 녹음하거나 자막으로 대체 가능. BGM은 잔잔한 무가사 트랙.

### ✅ 면적 자동취득 검증된 데모 주소 (2026-06-29 라이브 확인)
녹화 시 아래 주소를 쓰면 "주소 → 필지 면적 자동" 장면이 깔끔하게 나옵니다(필지 매칭 보장).
- **충북** 충청북도 충주시 가주동 483 → 5,487㎡
- **경북** 경상북도 안동시 임동면 대곡리 산 501-1 → 2,830㎡
- **충남** 충청남도 예산군 오가면 월곡리 31-4 → 3,358㎡
> ※ 도별로 하나씩 보여주면 "충북·경북·충남 3개 도 커버"가 자연스럽게 드러납니다.

## 강조 체크 (평가 배점 직결)
- **공공데이터·AI(30)**: 장면 4(팜맵)·5(공시지가·실거래·소득·KAMIS)·6(생성형 AI) 에서 데이터 출처를 자막으로 반드시 노출.
- **기술성(20)**: 장면 4 면적 자동 취득, 5 즉시 산출의 매끄러움.
- **독창성(20)**: 장면 5의 "결정론+AI 분리", 6의 관점별 리포트.
- **발전 가능성(20)**: 장면 8~9의 매칭→상담→대화로 이어지는 실거래 가능성.
- **시의성(10)**: 장면 2 통계.

## 촬영 팁
- 로딩(스피너)은 편집에서 컷 또는 1.5~2배속.
- 면적 자동취득은 **사전 준비의 검증된 주소**(충주 가주동·안동 대곡리·예산 월곡리)를 사용. 임의 주소로 "9개만 되는 게 아님"을 보이려면 **과수 지역 실주소를 녹화 전 직접 테스트**해 자동취득되는 것만 사용.
- (선택) 장면 4~5 사이에 **"시설·판로 입력(선택)"** 펼쳐 보유 시설·연 매출·판로 입력 → 시설 잔존가·영업권 반영 + **신뢰등급 C→B 상승**을 보여주면 "자료를 낼수록 정밀해지는 단계적 평가"가 드러남(공공데이터·기술성 가점). 시간상 생략 가능.
- 대시보드·매칭 카드의 면적은 `0.55ha (약 1,660평)`처럼 평수 병기 — 농가 친화 디테일로 짧게 비춰도 좋음.
- 마지막에 팀(김정훈·윤채원·유수민) 한 줄 크레딧.
