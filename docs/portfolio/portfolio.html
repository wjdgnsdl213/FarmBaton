문제 인식

농장은 멈추는데,
이어받을 통로가 없습니다

농촌 고령화는 역대 최고치를 경신하고, 들어오려는 청년농은 역대 최저로 줄었습니다.
세대 사이를 이어줄 다리가 없는 것이 가장 큰 문제입니다.

55.8%
65세 이상 농가인구 비율
(처음 55% 돌파 · 역대 최고)
50.8%
70세 이상 경영주 비중
(과반 · 약 49.5만 가구)
4,601가구
40세 미만 청년 경영주
(2020년 12,426 → 역대 최저)
출처: 통계청 「2024년 농림어업조사」

수십만 농가가 은퇴를 앞두고 있지만 후계자가 없고, 농장을 이어받으려는 청년농과 연결할 통로도 없습니다.

게다가 농장의 가치를 객관적으로 매기기 어렵습니다. 감정평가는 비용과 시간이 들고, 토지·과수·시설·판로가 얽힌 영농 기반은 더욱 그렇습니다.

그 결과 정보 비대칭으로 멀쩡한 영농 기반이 그대로 사라집니다. 팜바톤은 이 승계의 사각지대를 공공데이터로 잇습니다.

핵심 기능

농장주와 청년농,
양쪽 입장에서 설계했습니다

같은 공공데이터·가치평가 결과를 두 사용자의 언어와 목적에 맞춰 다르게 제공합니다.

◤ 농장을 넘기려는 고령 농가
농가 등록 → 인수 검토 리포트
  • 주소·작목·수령을 입력하면 팜맵으로 필지 면적을 자동 취득
  • 예상 연소득·토지·시설·영업권을 결합해 인수 검토가 범위와 신뢰등급(A~D) 산출
  • 농장주 관점 AI 리포트(매도 준비·시설 가치 어필)를 PDF로
  • 매칭 풀에 공개해 조건 맞는 청년농 후보를 추천받기
✓ 감정평가 없이 무료·수십 초로 내 농장 값을 가늠합니다
◤ 농장을 찾는 청년농
프로필 → 매칭 리스트
  • 희망 지역·작목·보유 자본·경력·정책자금을 입력
  • 6요인(지역·작목·자본·경험·승계·정책) 점수순으로 승계 가능 농장 추천
  • 청년농 관점 AI 리포트(자본 회수·정책자금·체크리스트)와 맞춤 지원사업 확인
  • 인앱 상담 신청 → 농장주 수락 시 양방향 채팅
✓ 조건에 맞는 승계 농장을 점수로 발견합니다

두 사용자가 보는 숫자는 같은 결정론적 가치평가 결과입니다. 생성형 AI가 이를 매도·인수 각자의 관점으로 번역해, 고령 농가와 청년농의 정보 격차를 좁힙니다.
