## [복붙용 본문]

팜바톤은 농림축산식품 및 국가 공공데이터를 결합해 농장의 인수 검토가 범위와 예상 연소득을 산출합니다. 활용한 공공데이터는 다음과 같습니다.

1. 팜맵(농경지 전자지도, 필지 경계·지목·면적 SHP) — 출처: V-World 공간정보 오픈플랫폼(국토교통부) 데이터 다운로드 / 원천: 농림수산식품교육문화정보원(농정원). 필지 데이터를 PostGIS에 적재해 입력 주소의 과수원 필지 면적·위치를 공간연산으로 자동 취득.

2. 주소 지오코딩(주소→좌표 변환) — 출처: V-World 공간정보 오픈플랫폼 주소 API(국토교통부, api.vworld.kr). 농장 등록 시 입력 주소를 좌표로 실시간 변환(필지 검색의 출발점).

3. 개별공시지가(토지특성 포함) — 출처: V-World 공간정보 오픈플랫폼(국토교통부) / 원천: 국토교통부. 시군구별 공시지가를 토지 기준가 산정에 활용.

4. 농산물 소득조사(작목별 10a당 소득·경영비·소득률; 사과·복숭아·포도) — 출처: 농촌진흥청(RDA) 「농산물소득조사」, 농사로(nongsaro.go.kr)·농식품 공공데이터포털. 작목별 예상 연소득 산정의 기준 계수로 활용.

5. 농산물유통정보 KAMIS 소매가격·평년가격 — 출처: 한국농수산식품유통공사(aT) KAMIS(kamis.or.kr). 과수 시세 및 평년가격(최근 5개년 중 최고·최저 제외 3개년 평균)으로 예상 소득의 시세 보정·시세 참고에 활용.

6. 토지(전·답·과수원) 실거래가 — 출처: 국토교통부 실거래가 공개시스템(rt.molit.go.kr). 공시지가 기반 토지 기준가의 시장가 보정에 활용.

7. 법정동코드 — 출처: 행정안전부, 공공데이터포털(data.go.kr). 주소 정규화 및 지역(시도·시군구) 매칭에 활용.

※ V-World(국토교통부 공간정보 오픈플랫폼)는 실시간 지오코딩 API와 적재용 데이터(팜맵·개별공시지가) 다운로드 양쪽에 활용해, 단일 공공 플랫폼을 다목적으로 깊게 활용했습니다. 지도 화면 표출에는 OpenStreetMap 타일을 사용하나 이는 공공데이터가 아닌 위치 시각화 용도입니다.
