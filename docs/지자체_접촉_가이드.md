- **담당자 찾기**: 시군 홈페이지 조직도에서 "귀농귀촌" 또는 "인력육성" 검색 → 담당 직통번호. 못 찾으면 대표번호에 "청년 창업농·후계농 담당자 부탁드립니다."

## 5. 미팅 아젠다 (15분) & 발표 사용 표현 기준

**아젠다**: ① 관내 주소로 라이브 데모 5분 (사전에 해당 주소 자동취득 테스트 + 프록시 가동 확인) → ② 제안 3가지 5분 → ③ 질의 5분.
