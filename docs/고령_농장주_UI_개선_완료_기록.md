# 고령 농장주 친화 UI 개선 완료 기록

작성일: 2026-07-24

## 목적

70대 이상 농장주도 도움 없이 농장을 등록할 수 있도록 입력 부담을 낮추고, 청년농 화면의 밀도와 역할을 유지한다.

## 완료 기능

### 1. 목적 중심 역할 선택

- 첫 화면에서 연령 대신 목적을 기준으로 선택한다.
  - `내 농장 정리` → 농장주 등록 흐름
  - `이어받을 농장 찾기` → 청년농 매칭 흐름
- 대표 이미지는 히어로에 보존하고, 역할 선택은 바로 아래의 짙은 초록 영역으로 분리했다.
- 카드의 번호·부가 설명·왼쪽 강조선을 제거해 제목과 시작 동작만 남겼다.

### 2. 농장주 등록 3단계

1. **농장 위치**: 주소 검색, 지도 확인, 면적 입력
2. **농장 정보**: 작목, 수령, 승계 방식
3. **추가 정보**: 시설·판로·매출 선택 입력, 리포트 산출

- 1단계에서는 주소와 위치 확인 또는 면적 직접 입력 전에는 다음 단계로 이동하지 않는다.
- 시설·판로·매출은 선택 입력이며, 생략해도 인수 검토 리포트를 받을 수 있다.
- 기존 가치평가 요청 데이터와 제출 로직은 유지했다.

### 3. 현재 위치로 주소 입력

- 농장 위치 단계에서 `현재 위치로 주소 입력` 버튼을 누를 때만 브라우저 위치 권한을 요청한다.
- 위치 허용 시 좌표를 지도에 표시하고 V-World 역지오코딩으로 주소 입력란을 채운다.
- 사용자는 지도 위치가 실제 농장과 맞는지 확인해야 한다. 현재 위치가 곧 농장 위치라고 단정하지 않는다.
- 위치 권한 거부, 주소 변환 실패, 지원하지 않는 기기에서는 지도 표시 또는 주소 직접 입력으로 이어진다.
- V-World 장애 시에는 사전 등록된 데모 좌표와 120m 이내일 때만 정적 주소 폴백을 사용한다. 그 밖의 위치에는 임의 주소를 채우지 않는다.
- 국내 V-World 프록시도 좌표→주소 요청을 지원하도록 확장했다.

### 4. 화면 설정과 재선택

- 상단의 `가 글씨 크게` 버튼으로 큰 글씨를 켜고 끌 수 있다.
- 설정은 현재 기기의 브라우저에 저장되어 다음 방문에도 유지된다.
- 모바일 메뉴의 `처음 화면에서 역할 다시 고르기`로 역할 선택 화면으로 돌아갈 수 있다.
- 이 동작은 화면 진입만 바꾸며, 로그인 계정의 농장주·청년농 권한이나 데이터는 바꾸지 않는다.

## 사용자 테스트 계획

실제 고령 농장주 참여 테스트는 아직 실시하지 않았다. 아래 절차로 최소 3명(65세 이상 농장주)을 대상으로 진행한다.

1. 안내 없이 첫 화면에서 본인 목적에 맞는 흐름을 선택하는지 관찰한다.
2. `글씨 크게`를 켜고 끄며 글자가 충분히 읽히는지 묻는다.
3. 농장주 등록에서 현재 위치 버튼을 눌러 권한 안내와 지도 확인 과정을 수행한다.
4. 위치 권한을 거부한 뒤 주소 직접 입력으로 계속 진행할 수 있는지 확인한다.
5. 3단계 입력에서 각 단계의 목적과 다음 버튼을 이해하는지 확인한다.
6. 선택 입력을 건너뛰고 리포트까지 도달하는지 확인한다.

기록 항목: 완료 여부, 막힌 단계, 도움 요청 횟수, 발화(혼란스러운 문구), 글자 크기 만족도.

통과 기준: 참여자 3명 중 2명 이상이 도움 없이 역할 선택과 1단계 위치 확인을 완료하고, 모든 참여자가 주소 직접 입력 대안을 인지한다.

## 운영 전 확인 사항

- 운영 환경에 `VWORLD_API_KEY` 또는 국내 `VWORLD_PROXY_URL`/`VWORLD_PROXY_TOKEN`이 설정되어야 일반 위치의 자동 주소 입력이 가능하다.
- V-World 키가 없거나 주소 변환이 실패해도 주소 직접 입력과 면적 직접 입력으로 농장 등록을 계속할 수 있어야 한다.
- 실제 사용자 테스트 결과는 이 문서의 ‘사용자 테스트 계획’ 아래에 참여자 식별 정보 없이 추가 기록한다.
