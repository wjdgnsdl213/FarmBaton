## 1. 공모아이템요약 (4,000자 이내)

## [복붙용 본문]

■ 아이템명: 팜바톤(FarmBaton) — 고령 농가의 농장(농지·작목·시설·판로)을 청년농에게 잇는 농장 승계 진단·매칭 플랫폼(모바일 퍼스트 반응형 웹).

■ 배경·문제: 65세 이상 농가인구 비율이 55.8%로 사상 최고이고 70세 이상 경영주가 과반(49만여 가구)인 반면, 40세 미만 청년 경영주 농가는 2020년 1만 2,426가구에서 2024년 4,601가구로 4년 만에 약 1/3로 급감했습니다(통계청 농림어업조사). 수십만 농가가 후계자 없이 은퇴를 앞두고 있으나, 농지·과수·시설·판로로 이어진 '영농 기반 전체'를 객관적으로 정리하고 들어올 청년농과 연결할 통로가 부족합니다. 기존 부동산 중개는 토지 매매 중심이고, 전문 평가 의뢰는 비용·시간 부담이 커 초기 승계 검토에 활용하기 어렵습니다.

■ 해결(서비스 개요): 팜바톤은 농장주가 주소와 기본 정보를 입력하면 팜맵·개별공시지가·토지 실거래가·농산물 소득조사·KAMIS 시세 등 공공데이터를 결합해 '인수 검토가 범위(참고용 추정)'와 예상 연소득을 자동 산출하는 서비스입니다. 생성형 AI는 산출된 수치를 바꾸지 않고, 농장주·청년농 관점별 리포트로 쉽게 설명합니다. 청년농은 희망 지역·작목·자본 조건을 입력해 매칭 점수가 높은 승계 가능 농장을 추천받고 상담·채팅으로 이어질 수 있습니다.

■ 핵심 기능 2가지:
① 농가 등록 → 인수 검토 리포트(농장주): 주소 입력 시 V-World 지오코딩과 팜맵 공간연산으로 필지 면적·위치를 자동 취득하고, 작목(사과·복숭아·포도)·수령·승계 방식을 입력하면 결정론적 가치평가 엔진이 ▲예상 연소득(소득조사 10a당 소득×면적×수령계수×KAMIS 시세보정) ▲토지 기준가(공시지가×면적, 실거래가 보정) ▲시설 잔존가 ▲영업권을 합산해 인수 검토가를 min~max 범위와 신뢰등급(A~D)으로 제시합니다. 시설·판로·매출을 선택 입력할수록 정밀도가 높아지고 신뢰등급이 상승합니다.
② 청년농 프로필 → 매칭 리스트(청년농): 희망 지역·작목·자본·경력·정책자금·승계 선호를 입력하면 6요인(지역·작목·자본·경험·승계·정책) 스코어링(100점)으로 승계 가능 농장을 점수순 추천하고, 맞춤 지원사업·관점별 PDF 리포트 확인 후 인앱 상담 신청 → 농장주 수락 시 채팅으로 승계 논의를 시작합니다.

■ 공공데이터·AI 활용: 팜맵·개별공시지가·주소 지오코딩(이상 V-World/국토교통부), 농산물 소득조사(농촌진흥청·농사로/농식품 공공데이터포털), KAMIS 소매·평년가격(aT), 토지 실거래가(국토교통부), 법정동코드(행정안전부)를 활용합니다. 핵심 설계는 '숫자는 알고리즘, 설명은 AI'의 역할 분리입니다. 금전 추정치와 매칭 점수는 100% 결정론적 함수로 산출해 환각으로 인한 잘못된 금액 제시를 차단하고, 생성형 AI(Claude)는 동일 수치를 고령 농가·청년농 각자의 언어와 관점으로 번역하는 개인화 설명에만 사용합니다.

■ 차별성·독창성: ▲토지뿐 아니라 과수 수령·시설 잔존가·영업권까지 반영한 승계 특화 가치평가 모델 ▲결정론 계산+생성형 AI 설명의 분리 아키텍처로 신뢰성과 개인화 동시 달성 ▲공공데이터 국외반출 제약(V-World의 해외 IP 차단)을 국내 회선 우회 프록시로 해결한 실전 엔지니어링 ▲청년창업농 정책자금 한도 내 인수 가능 농장을 우대하는 양면 매칭이 특징입니다.

■ 추진·기대효과: 충북·경북·충남 과수 3작목으로 검증한 MVP를 전국·전작목으로 확장하고, 지자체 농업기술센터·농지은행·청년농 지원사업과 연계한 B2G/B2B2C로 사업화합니다(2026년 하반기 개인 사업자등록 목표). 승계 검토 진입장벽을 비용·수일 단위의 외부 의뢰에서 무료·수십 초의 사전 검토로 낮추고, 사라지던 영농 기반을 데이터로 다음 세대에 이어 농업 세대전환·지역소멸 대응·식량안보에 기여합니다. 전 기능이 production(farmbaton.vercel.app)에서 실제 동작합니다.

## 2. 공모 서비스 요약 (4,000자 이내)

## [복붙용 본문]

팜바톤(FarmBaton)은 고령 농가가 보유한 농장 자산과 영농 기반을 청년농에게 연결하는 농장 승계 진단·매칭 서비스입니다. 서비스의 핵심은 농장주에게는 “내 농장을 어느 정도 조건으로 넘길 수 있는지”를 공공데이터 기반으로 빠르게 보여주고, 청년농에게는 “내 자본·작목·지역 조건에 맞는 승계 가능 농장이 어디인지”를 점수순으로 추천하는 것입니다. MVP는 충북·경북·충남 3개 도와 사과·복숭아·포도 3개 과수 작목에 집중해 실제 데모 가능한 범위로 구현했습니다.

서비스는 두 가지 주요 흐름으로 구성됩니다.

첫째, 농장주용 “농가 등록 → 인수 검토 리포트” 흐름입니다. 농장주는 회원가입 후 주소를 입력하고 작목, 과수 수령, 희망 승계 방식, 시설·판로·매출 정보를 단계적으로 입력합니다. 주소는 V-World 지오코딩을 통해 좌표로 변환되고, 팜맵 기반 공간연산으로 필지 위치와 면적을 자동 취득합니다. 이후 개별공시지가, 토지 실거래가 적재본, 농산물 소득조사, KAMIS 시세, 시설 표준단가, 수령계수 등을 결합해 예상 연소득과 “인수 검토가 범위(참고용 추정)”를 산출합니다. 결과 화면은 토지 기준가, 시설 잔존가, 영업권, 예상 연소득, 신뢰등급(A~D), 산출 근거를 함께 보여주며, 단일 가격이 아니라 min~max 범위로 제시합니다. 시설 자료, 매출 자료, 판로 정보가 많을수록 신뢰등급이 올라가도록 설계해 농장주가 추가 자료를 제출할 유인을 제공합니다.

둘째, 청년농용 “프로필 등록 → 매칭 리스트” 흐름입니다. 청년농은 희망 지역, 희망 작목, 보유 자본, 영농 경력, 정책자금 활용 여부, 선호 승계 방식을 입력합니다. 팜바톤은 지역 적합도, 작목 일치도, 자본 충족도, 경험 수준, 승계 방식 적합도, 정책자금 조건 등 6개 요인을 100점 만점으로 계산해 승계 가능 농장을 점수순으로 추천합니다. 청년농은 농장별 예상 연소득, 인수 검토가 범위, 시설 현황, 맞춤 지원사업, 관점별 리포트를 확인한 뒤 상담을 신청할 수 있습니다. 농장주가 상담 요청을 수락하면 인앱 채팅으로 이어져 실제 승계 논의를 시작할 수 있습니다.

서비스의 기술 구조는 신뢰성과 데모 안정성을 우선해 설계했습니다. 백엔드는 FastAPI와 PostgreSQL/PostGIS 기반이며, 팜맵 SHP 등 공간데이터는 EPSG:4326으로 통일해 저장합니다. 프런트엔드는 React(Vite)와 Leaflet 지도를 사용한 모바일 퍼스트 반응형 웹입니다. 외부 API 장애가 발생해도 서비스가 멈추지 않도록 모든 외부 호출에는 정적 폴백을 두었고, 실거래가는 API 실시간 호출 없이 DB에 적재한 정적 데이터만 사용합니다. 단가·계수·기준값은 코드에 하드코딩하지 않고 DB 기준 테이블을 참조합니다.

AI는 설명 생성에만 사용합니다. 금전 추정치와 매칭 점수는 모두 결정론적 Python 함수로 계산해 동일 입력에 동일 결과가 나오도록 했습니다. 생성형 AI(Claude)는 계산된 수치를 바꾸지 않고, 농장주에게는 승계 준비와 농장 가치 설명 관점으로, 청년농에게는 자본 회수 가능성·주의사항·정책자금 활용 관점으로 리포트를 풀어 쓰는 역할만 담당합니다. 즉, “숫자는 알고리즘, 설명은 AI”로 분리해 환각 위험을 줄이고 설명의 이해도를 높였습니다.

팜바톤은 단순 농지 매물 서비스가 아니라 농지·과수·시설·판로를 묶은 영농 기반 전체의 승계를 다룹니다. 고령 농가는 복잡한 데이터와 평가 과정을 거치지 않고도 무료로 사전 검토 리포트를 받을 수 있고, 청년농은 막연한 탐색 대신 자신의 조건에 맞는 농장을 우선순위로 확인할 수 있습니다. 현재 production 환경(farmbaton.vercel.app)에서 농가 등록, 리포트 생성, 청년농 매칭, 상담 신청, 채팅까지 핵심 흐름이 실제 동작하도록 구현되어 있습니다.
